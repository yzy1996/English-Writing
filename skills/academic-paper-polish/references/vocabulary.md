# Vocabulary Reference for Academic Paper Writing

## Abbreviations & Latin

| Abbreviation | Full Form | Meaning |
|---|---|---|
| e.g. | exempli gratia | for example |
| i.e. | id est | in other words |
| a.k.a. | also known as | 又称 |
| w.r.t. | with regard to | 关于 |
| etc. | et cetera | and so forth |

Latin terms (always *italicize*): *per se*, *de facto*, *vice versa*, *pseudo*, *post hoc*, *ad-hoc*

## Positive Evaluation — Abstract Concepts

Use when praising a research area, paradigm, or achievement:

| Term / Phrase | Notes |
|---|---|
| achieved unprecedented xx | breakthrough framing |
| shed light on these hard problems | illuminating insight |
| made tremendous progress | strong momentum |
| phenomenal success | hyperbolic, use sparingly |
| the seminal / notable / pioneering work | attributing credit to prior work |
| intriguing / appealing | interesting, draws attention |
| unprecedented | never done before |
| indispensable / exclusive | essential / unique |
| gain a surge of interest | becoming trendy |
| imperative / extraordinary | critical / exceptional |
| prominently / spectacularly | clearly / strikingly |
| leap forward | major advance |
| fruitful / tailored | productive / customized |
| overwhelming evidence | very strong support |
| spark wide interest | inspiring the community |

---

## Positive Evaluation — Concrete Methods

Use when praising your own method or a component:

| Term / Phrase | Notes |
|---|---|
| eliminates the unidirectional bias | addresses a specific limitation |
| ameliorate / alleviate | improve / ease (a problem) |
| xx will favor yy | beneficial to |
| synergistic to other approaches | works together with others |
| benefiting from the maturity of... | building on proven foundations |
| modifications are highly complementary | mutually reinforcing |

---

## Negative Evaluation — Critiquing Baselines

Use when pointing out limitations of existing methods:

| Term / Phrase | Notes |
|---|---|
| shortcoming / drawback / deficiency | general weakness |
| undesirable | unfavorable outcome |
| repercussion / minimal side effect | negative consequence |
| detrimental / adversely affects | harmful impact |
| hamper / hinder / hindrance | obstruct / slow down |
| deteriorate | make worse |
| degrade / damage / decrease / drop | performance degradation |
| inferior | lower quality |
| vulnerable | susceptible to failure |
| notoriously | famously bad |
| be confined to | limited to only |
| lagged far behind | significantly behind SOTA |

## Difficulty Vocabulary

Use when describing why a problem is hard:

| Term / Phrase | Notes |
|---|---|
| solve / fix / address / tackle challenges | action verbs |
| a remedy to this problem | a solution |
| xx struggle to do sth | baseline fails |
| elusive | hard to grasp / achieve |
| a prominent difficulty / hurdle | standout challenge |
| an inevitable problem | unavoidable |
| prohibitively high cost | too expensive to run |
| the core difficulty | central bottleneck |
